# word-corners-tests

## Start

1.
```bash
npm install
```

2. 
Run tests in terminal (headless)
```bash
npm test
```

Run tests in browser
```bash
npm run test-ui
```

## Github Actions

We run the tests using Github Actions. The tests are run on every push or pull request to the main branch.
